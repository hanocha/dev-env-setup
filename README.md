# dev-env-setup

Setting up development environment for Ubuntu, running on WSL2.


## targets

- Ubuntu 22.04
- ~~macOS~~

## usage

You should only run a command below!

```
$ sudo ./bin/setup
```

## Packages to be installed

See also: `main.rb`

- Powerline fonts
- fish, fisherman, bobthefish theme
- go, ghq, peco
- rbenv
- volta (nodejs version manager)
